# java-sample-app
## sample1
Dynamoへの登録・検索をするサンプルAPI

### 実行コマンド

#### ネットワーク作成
docker network create test-network

#### キャッシュ使用しないでビルド
docker-compose build --no-cache

#### 起動
docker-compose up
